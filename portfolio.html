Javascript Calculator

Created a javascript calculator similar to the Apple calculator. Uses logic and math functions to make calculations. One notable difference is that this calculator performs the modulus function which is useful and common in programming.

Weather Map Site

By using the Google Maps API and a weather API, we are able to find the 3-day forecast for any location by dragging and dropping the pin on the website. A search function is also utilizes Google maps to geocode any location entered by the user.

Simple Simon

This game uses javascript to create a copy of the popular Simon game from the 80s. A random sequence is generated and the user copies that sequence by clicking on the corresponding squares.

Whack-a-Weenie

A variation on a whack-a-mole style game where the user tries to click on as many "weenies" (dachshunds) to get a high score. A surprise pops up when a user gets the new high score.
